# Tabs

Allow navigation between groups of content that are related.

`import Tabs from "design-system/patterns/tabs"`

Data should be provided in the following format:

```js
const data = [
  {
    tab: `Tab 1`,
    content: `Content 1`,
  },
  {
    tab: `Tab 2`,
    content: `Content 2`,
  },
]
```

Each data item should contain the `tab` and `content` properties in order to render Tabs.

## Custom components as Tab content

You can provide anything you want within the tab content.

```js
const data = [
  {
    tab: `Tab 1`,
    content: (
      <List>
        <ListItem iconName="check" variant="gray">
          content 1
        </ListItem>
        <ListItem iconName="check" variant="gray">
          content 1
        </ListItem>
      </List>
    ),,
  },
  {
    tab: `Tab 2`,
    content: (
      <List>
        <ListItem iconName="check" variant="gray">
          content 2
        </ListItem>
        <ListItem iconName="check" variant="gray">
          content 2
        </ListItem>
      </List>
    ),
  }
]
```
